# Starter Server
Node.js + Express + Typescript + Prisma


### `installation`

Just install packages by running

```
npm install
```

### `development`

Start your server application with autoReload enabled.

```
npm run dev
# or
yarn dev
```

### `build`

Build your server.

```
npm run build
# or
yarn build
```
